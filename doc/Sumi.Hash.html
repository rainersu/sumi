(static) uuid1(nodeopt, clockseqopt, msecsopt, nsecsopt) → {string}

生成一个符合 RFC4122,v1 规则的 UUID(Universally Unique IDentifier) 。
Parameters:
Name Type Attributes Description
node array <optional>
 作为 6 个字节数组的 Node Id。默认随机产生
clockseq number <optional>
 取值范围为 0 - 0x3fff 的 RFC clock sequence
msecs number | date <optional>
 自 unix Epoch 开始累计的时间，单位为毫秒。默认为当前时间
nsecs number <optional>
 取值范围为 0-9999 的附加时间, 单位为 100-nanosecond 。如果 msecs 未指定则被忽略
