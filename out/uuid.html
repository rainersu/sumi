(static) v1(optionsopt, bufferopt, offsetopt) → {string|buffer}

生成一个符合 RFC4122,v1 规则基于时间戳的 UUID(Universally Unique IDentifier)。
Parameters:
Name Type Attributes Default Description
options Object <optional>
Properties
Name Type Description
node array 作为 6 个字节数组的 Node Id。默认随机产生
clockseq number 取值范围为 0 - 0x3fff 的 RFC clock sequence
msecs number | date 自 unix Epoch 开始累计的时间，单位为毫秒。默认为当前时间
nsecs number 取值范围为 0-9999 的附加时间, 单位为 100-nanosecond 。如果 msecs 未指定则被忽略
buffer array | buffer <optional>
 要将生成出的 UUID 写入的数组或缓冲字节流
offset number <optional>
 0 写入 buffer 的开始处索引

  • uuid.v4
Returns:
Type
string | buffer
Example
console.log(uuid.v1());

console.log(uuid.v1({
    node: [0x01, 0x23, 0x45, 0x67, 0x89, 0xab],
    clockseq: 0x1234,
    msecs: new Date('2011-11-01').getTime(),
    nsecs: 5678
}));

console.log(uuid.v1(null, new Array(32), 0));   // [02 a2 ce 90 14 32 11 e1 85 58 0b 48 8e 4f c1 15])
